绝大部分可以书写尺寸的地方，都可以书写百分比

百分比是一个相对单位，其相对于元素的[[包含块]]

| css属性   | 百分比相对于     | 备注                 |
| ------- | ---------- | ------------------ |
| width   | 包含块的**宽度** |                    |
| height  | 包含块的**高度** | 参考系高度受本身宽度影响时，设置无效 |
| padding | 包含块的**宽度** |                    |
| border  | 包含块的**宽度** |                    |
| margin  | 包含块的**宽度** |                    |
